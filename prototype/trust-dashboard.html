How Trust Scoring Works
📝
Agents Register
Software agents register and receive a cryptographic token. Every agent starts at 0.50 — the score for "unknown."
📈
Behavior Builds Trust
Each verified interaction increases the SubScore. More queries, more history, more data for businesses to evaluate.
⚠️
Incidents Lower SubScores
Third parties report problems. Submantle records them — it never detects. Like a credit bureau: banks report missed payments.
✅
Businesses Decide
Submantle never blocks or enforces. It provides the score. Each business sets their own threshold — just like Visa.
The Formula
SubScore = (queries + 1) / (queries + incidents + 2)
Beta Reputation with Laplace smoothing. Pure math — no ML, no inference. Deterministic, auditable, outside EU AI Act scope.
